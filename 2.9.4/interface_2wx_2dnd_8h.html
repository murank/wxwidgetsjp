Enumerator:
wxDrag_CopyOnly 

Allow only copying.

Allow moving too (copying is always allowed).

wxDrag_DefaultMove 

Allow moving and make it default operation.

Enumerator:

Error prevented the D&D operation from completing.

wxDragNone 

Drag target didn't accept the data.

データの移動に成功した。(MSW のみ)

wxDragLink
